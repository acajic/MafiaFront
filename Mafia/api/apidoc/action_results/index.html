GET /action_results
[ADMIN] Query Action Results.

ADMIN

Query Action Results.

Supported Formats

json

Params

Param name Description
auth_token
required

Auth Token is obtained via /login and destroyed via /logout endpoint.


Value:

Must be String
result_json
optional

Action Results whose result_json contains this string.


Value:

Must be String
action_ids
optional

Action Results that were produces by one of these actions. Example:

exposemafia.com:3000/action_results?action_ids[]=62&action_ids[]=63

Value:

Must be an array of Integer
action_result_type_ids
optional

Action Results that are of these types. Example:

exposemafia.com:3000/action_results?action_result_type_ids[]=4&action_result_type_ids[]=5

Value:

Must be an array of Integer
is_automatically_generated
optional

Is the Action Result automatically generated? If 'true', the Action Result is authentic. If 'false', the Action Result is faked by the user.


Value:

Must be one of: true, false.
city_ids
optional

Action Results from the specified cities. Example:

exposemafia.com:3000/action_results?city_ids[]=6&city_ids[]=5

Value:

Must be an array of Integer
city_name
optional

Action Results whose name contains this string.


Value:

Must be String
day_number_min
optional

Action Results that are produced on or after the specified game day.


Value:

Must be Integer
day_number_max
optional

Action Results that are produced on or before the specified game day.


Value:

Must be Integer
resident_ids
optional

Action Results produces specifically for specified residents. Example:

exposemafia.com:3000/action_results?resident_ids[]=16&resident_ids[]=17&resident_ids[]=21

Value:

Must be an array of Integer
for_all_residents
optional

Return only the Action Results that are intended for all the residents in the city ('true'), or the ones that are intended only for specific residents ('false').


Value:

Must be one of: true, false.
resident_username
optional

Action Results that are intended for a resident whose name contains this string.


Value:

Must be String
role_ids
optional

Action Results for the specific roles. Example:

exposemafia.com:3000/action_results?role_ids[]=3&role_ids[]=2

Value:

Must be an array of Integer
deleted
optional

Return only the Action Results were deleted by the user ('true'), or the ones that weren't deleted by the user ('false').


Value:

Must be one of: true, false.
created_at_min
optional

Only Action Results that were created after this timestamp will be returned.


Value:

Must be Timestamp - a number representing number of seconds since 1970-01-01T00:00:00.
created_at_max
optional

Only Action Results that were created before this timestamp will be returned.


Value:

Must be Timestamp - a number representing number of seconds since 1970-01-01T00:00:00.
updated_at_min
optional

Only Action Results that were updated after this timestamp will be returned.


Value:

Must be Timestamp - a number representing number of seconds since 1970-01-01T00:00:00.
updated_at_max
optional

Only Action Results that were updated before this timestamp will be returned.


Value:

Must be Timestamp - a number representing number of seconds since 1970-01-01T00:00:00.
page_index
optional

Page index.


Value:

Must be Integer
page_size
optional

Page size.


Value:

Must be Integer
