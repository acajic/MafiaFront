[ADMIN] Query Residents

ADMIN

Supported Formats

json

Params

Param name Description
auth_token
optional

Auth Token is obtained via /login and destroyed via /logout endpoint.

optional

Residents with there ids will be returned. Example:

exposemafia.com:3000/residents?user_ids[]=6&user_ids[]=5

Value:

Must be an array of Integer
username
optional

Residents that belong to users whose usernames contain this string.


Value:

Must be String
name
optional

Residents whose names contain this string.


Value:

Must be String
city_ids
optional

Residents to at least one of these cities. Example:

exposemafia.com:3000/residents?city_ids[]=6&city_ids[]=5

Value:

Must be an array of Integer
city_name
optional

Residents whose cities' names contain this string.


Value:

Must be String
role_ids
optional

Residents whose true game roles are one of the specified. Example:

exposemafia.com:3000/residents?role_ids[]=2&role_ids[]=3&role_ids[]=4

Value:

Must be an array of Integer
saved_role_ids
optional

Residents whose saved game roles are one of the specified. Example:

exposemafia.com:3000/residents?saved_role_ids[]=2&saved_role_ids[]=3&saved_role_ids[]=4

Value:

Must be an array of Integer
role_seen
optional

Residents have logged in into the game at least once. They are the ones who have seen their game role.


Value:

Must be one of: true, false.
alive
optional
