Teller plays on board with the Citizens.
He has a special ability to count the votes of public voting.
After a day ends, Teller is presented with information which resident received how many votes during public voting phase. The report does not include information about who voted for whom.

If your Special tab is set up so that it displays that you have unlimited Request Vote Count actions, than an action of this type will automatically be submitted for you on every game day. Otherwise, you have to manually request the vote count in order to get it on the following game evening.
