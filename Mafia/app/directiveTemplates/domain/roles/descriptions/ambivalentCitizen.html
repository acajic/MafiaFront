Ambivalent Citizen is a role similar to a basic Citizen.
Like the ordinary Citizen, Ambivalent Citizen's only ability is to vote who is going to be executed at the end of each day.
The only thing difference is that, in case of an ordinary citizen, at the end of the day phase, only the last vote action is taken into consideration.
When ambivalent citizen is casting votes, all votes targeting distinct residents are accepted.
Ambivalent residents can vote variable number of times, but each time for a different resident. This means that a resident with this role can submit [0..N] valid votes each day, with N being the number of players in the game.
If Ambivalent Citizen votes for the same resident multiple times in the same day, these duplicates count as a single vote.
