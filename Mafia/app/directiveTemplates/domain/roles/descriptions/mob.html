Mafia is a basic mafia role.
As any other mafia role, they can see who else belongs to the mafia.
Also, as any other mafia role, they can participate in mafia voting. A resident that is selected during mafia voting will be killed during the following night.

In order for mafia voting to be successful, strictly more than half of living mafia members must vote for the same player.
For example, if there are three living mafia members, two votes for the same player are necessary to kill them. If there are only two living mafia members, both of them must vote for the same player for mafia assasination to be successful.
