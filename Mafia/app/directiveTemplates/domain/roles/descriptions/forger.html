Forger is the same role as the Elder, only is sided with the Mafia.
Forger can forge public votes. Therefore Forger can vote multiple times in a single day, but not for the same person.
All votes targeting distinct residents are accepted.
A forger can vote variable number of times, but each time for a different resident. This means that a resident with this role can submit [0..N] valid votes each day, with N being the number of players in the game.
If a Forger votes for the same resident multiple times in the same day, these duplicates count as a single vote.

Also, as any other mafia role, Forger can participate in mafia voting. A resident that is selected during mafia voting will be killed during the following night.
